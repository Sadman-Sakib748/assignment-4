Question-3: What is event delegation in the DOM?

a JavaScript technique where you attach a single event listener to a parent element, allowing it to handle events triggered on any of its child elements

Question-4: What is event bubbling in the DOM?

the behavior where when an event occurs on an element, it not only triggers an event handler on that element but also propagates upwards through the DOM hierarchy, triggering event handlers on its parent elements until it reaches the topmost element (usually the document
